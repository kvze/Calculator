##Calculator

Небольшое веб‑приложение‑калькулятор на чистом JavaScript, HTML и CSS. Поддерживает базовые арифметические операции и работу с процентами.

## Возможности

- **Базовые операции**: сложение, вычитание, умножение, деление  
- **Проценты**: вычисление процентов от числа и в выражениях  
- **Изменение знака**: кнопка `+/-`  
- **Десятичная точка**: ввод дробных чисел  
- **Удаление**: кнопка backspace для поcимвольного удаления  
- **Очистка**: кнопка `AC` для сброса текущего ввода

## Технологии

- **HTML**: разметка интерфейса (`index.html`)
- **CSS**: оформление калькулятора (`css/style.css`)
- **JavaScript**: логика калькулятора в виде класса `Calculator` (`js/script.js`)
